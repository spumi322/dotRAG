Moving and updating data between databases, making sure everything stays organized and up-to-date during changes or upgrades.
### Data Migration Techniques:

1. **Database Dump and Restore:**
    
    - **Process:** Export data from the source database into a file (dump), and then import it into the destination.
2. **ETL (Extract, Transform, Load):**
    
    - **Process:** Extract, transform, and load data between systems, often used in data warehousing.
3. **Change Data Capture (CDC):**
    
    - **Process:** Captures changes made to the source data since the last migration, allowing incremental updates.
4. **Data Synchronization Tools:**
    
    - **Process:** Use tools to synchronize data between databases in real-time or on a schedule.
5. **Direct Database-to-Database Transfer:**
    
    - **Process:** Connect to both source and destination databases and transfer data directly.
